# 🏠 House Price Prediction Using Machine Learning

## 📌 Project Overview

House Price Prediction is a Machine Learning web application that estimates the price of a house based on various property features such as area, number of bedrooms, bathrooms, parking spaces, number of floors, furnishing status, and other amenities. The project uses a Linear Regression model trained on historical housing data to predict house prices accurately.

The application provides an easy-to-use web interface built with Flask, allowing users to enter house details and instantly receive an estimated property price.

---

## 🎯 Objective

The main objective of this project is to build a machine learning model that can predict house prices based on different housing features. It demonstrates the complete machine learning workflow, including data preprocessing, model training, evaluation, and deployment through a Flask web application.
